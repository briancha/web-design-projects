Course Description

Cascading Style Sheets (CSS3) - This course presents an introduction to Cascading Style Sheets (CSS) and their applications to the design and creation of web sites. Topics incluse CSS syntax, selectors. positioning, and internal and external style sheets

Learning Objectives

Upon completion of this course, the students will be able to:

  1. create cascading style sheets using CSS selectors, style attributes, and div and span.
  2. create cascading style sheets using a variety of font attributes.
  3. compose and manage child versus sibling and the box model style sheets.
  4. create CSS styling for colors, backgrounds, table, lists, and links.
  5. create CSS design using relative and absolute positioning, blocks, and clipping elements.
  6. create CSS for various devices sizes, printed and/or other media.
  7. examine cascading and inheritance, external style sheets, and the difference between tables and css.
