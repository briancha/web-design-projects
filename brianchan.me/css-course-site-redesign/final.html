Final Project

The final project for this course will be to redesign this course site.

The students will be provided with all HTML, java script and book image files. It will be up to the student to design the site in any manner they wish.

The only rules are the students many not modify the HTML files. Care has been made to provide sufficient selectors to the content to allow the user to configure the site in a virtually endless number of possibilities. That said, if you have something you would like to do that requires modification to the HTML files, please contact me first for approval.

Any additional banner or other images, java script implementation, etc., will be the responsibility of the student.
